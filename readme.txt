Hanu-Droid framework allows you to convert your word press blog in to a native Android app.

== Description ==
Hanu-Droid plugin allows you to convert your word press blog into an Android Native app for free. 
After you install the plugin, visit: http://hanu-droid.varunverma.org for more details and to get your free Android app. 

With this plugin, you can send push notifications to your app users whenever you add / update any post. To send push notifications, go to the settings - Hanu-Droid Settings and Send notifications.

== Installation ==

The installation is very easy. Just follow the below steps:

1. Upload Hanu-Droid folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
